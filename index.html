影像分割模型梯度加權類別活化映射分析

Image Segmentation Model Gradient-Weighted Class Activation Mapping Analysis

摘要

醫學影像分析領域中，心臟肌肉分割對疾病的診斷與治療具有重要意義。然而，深度學習模型的黑箱特性使醫療人員難以理解和信任其決策過程，因此需要有效的可視化方法來解釋模型的內部機制。本研究嘗試將 Grad-CAM 應用於三維醫學影像分割模型，以預訓練的 InceptionNeXt-ResCBAM 進行實驗，生成熱力圖揭示模型推論時的關注區域，並分析其在三維影像分割任務中的表現與解釋能力。本研究貢獻有二，其一，將 Grad-CAM 核心計算流程導入 InceptionNeXt-ResCBAM，分析其應用於三維醫學影像分割的適用性與限制。其二，探索從 Grad-CAM 的技術導入到生成熱力圖的過程中所遇到的挑戰與解決策略，包括特徵圖大小不符、記憶體需求高、模型輸入與目標層選擇等問題，並提出未來改進方向。為可解釋人工智慧在三維醫學影像分析領域的應用提供有價值的經驗與參考。

結果

熱力圖疊加於原始資料之結果。部分圖片亮區集中於影像的邊緣或背景，未能有效覆蓋心臟肌肉的主要區域。部分圖片則有大範圍亮區覆蓋，未能正確集中於心臟肌肉區域。顯示Grad-CAM 執行上可能有影像尺寸、滑動窗口設定等問題，導致生成結果不如預期。

執行步驟

Colab 內有程式執行步驟，執行前請依照右圖的結構在自己的雲端硬碟手動建立資料夾。並將要進行推論與熱力圖生成的影像放到 image 資料夾中，infer 資料夾則是用於儲存推論結果。

Step 1: 將程式碼從 GitHub clone 到工作階段中並安裝所需套件，建立推論所需資料夾。
Step 2: 下載權重檔。
Step 3: 將工作階段連接到自己的雲端硬碟。
Step 4: 執行推論，儲存 Grad-CAM 計算結果。
Step 5: 將 Grad-CAM 計算出的熱力圖逐切片疊加於原始影像並儲存成 png 檔案。
